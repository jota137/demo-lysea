# Lyséa Web Demo

Demo estática de la web de Lyséa para presentación.

## Publicar en GitHub Pages

1. Crear un repositorio nuevo en GitHub.
2. Subir el contenido de esta carpeta, no la carpeta completa:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `.nojekyll`
   - `assets/`
3. Entrar en `Settings > Pages`.
4. En `Build and deployment`, elegir:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guardar.

GitHub generara una URL parecida a:

`https://usuario.github.io/nombre-del-repositorio/`

## Nota

Esta es una demo HTML/CSS/JS. Para Shopify, el diseno se tendria que adaptar a tema Liquid, productos, colecciones, carrito y pagos.
